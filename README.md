# Testing Project

## Project Setup/Compile

    Make sure to run NPM Install prior to running the tests
    Load Ganache GUI App or ganache-cli
    Compile and Migrate contracts
    
    ``` 
    truffle compile
    truffle migrate
    ``` 

    To deploy contracts to the rinkeby test network

    ``` 
    truffle migrate --network rinkeby
    ``` 

## Running Tests
The project contains 3 test files. Run truffle test to execute.

- TestERC721Mintable -- Tests core functionality of ERC721 implementation
- TestSolnSquareVerifier -- Tests Add Solution and Mintiable Functionality 
- TestSquareVerifier -- Test Valid/Invalid Proofs using the verifer.sol Zokrates contract

    ``` 
    truffle test
    ```
